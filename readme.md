# 头歌平台作业自动完成脚本

## 本应用基于python和chrome拓展程序实现自动完成头歌平台的编程题目。

## 本应用仅供学习交流使用，如您使用本应用造成账号封禁、处分、退学等后果，开发者不承担任何责任！

## 使用步骤：
先启动python文件，再进入chrome://extensions/添加未打包的拓展程序。
在页面先点击链接，后点击获取题目，点击代码编译页面，等候代码自动完成。
代码需手动提交，只在[https://www.educoder.net/tasks/](https://www.educoder.net/tasks/)生效。

## 注意事项：
1. 请勿使用搜狗输入法等第三方输入法。这些输入法会在代码输入过程中自动切换中英文，使代码无法正确输入。请务必使用windows自带的微软拼音输入法。
2. 代码输入过程中请勿切换界面。如需停止代码输入，请将鼠标快速移动至屏幕左上角。

## 20251102更新内容：
